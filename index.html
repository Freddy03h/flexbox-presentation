order

À appliquer sur les flex-item, par defaut la valeur est à 0.


.item:nth-child(5) {
    order: -1;
}
.item:nth-child(2) {
    order: 1;
}

order

L'ordre dans le DOM défini l'ordre pour les flex-items de même order


.item:nth-child(4) {
    order: 1;
}
.item:nth-child(2) {
    order: 1;
